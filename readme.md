This is an interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**.  
The game allows the user to play against the computer, keeps track of **wins, losses, and draw matches**, and also provides a **New Game** button to reset the game anytime.

This project is my **first step into the Web Development domain**, helping me understand core JavaScript concepts and DOM manipulation.

---

## Features

- User vs Computer gameplay
- Random computer choice generation
- Real-time result display (Win / Lose / Draw)
- Score tracking for:
  - User score
  - Computer score
  - Draw matches count
- **New Game button** to reset scores and start fresh
- Interactive and responsive UI
- Beginner-friendly and clean JavaScript logic

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation & Events)

---

## How to Play

1. Click on **Rock**, **Paper**, or **Scissors**.
2. The computer randomly selects its move.
3. The game result is displayed instantly.
4. Scores for user, computer, and draw matches are updated.
5. Click **New Game** to reset all scores and start again.

---

## Game Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- If both choices are the same, the round is declared a **Draw**
